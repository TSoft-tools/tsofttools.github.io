# TSoft Tools

Профессиональные мобильные инструменты для инженеров и профессий связаных с расчётами конструкций.

## Приложения

### Жестянщик PRO

Мобильное приложение для:

- расчета развёрток;
- расчёта стоимости фасонных элементов;
- создания оптимального раскроя плоского листа;
- построения чертежей;
- создания PDF-документов;
- инженерных вычислений;
